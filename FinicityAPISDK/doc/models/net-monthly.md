
# Net Monthly

## Structure

`NetMonthly`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `month` | `bigint` | Required | Timestamp for the first day of this month |
| `net` | `number` | Required | Total income during the given month, across all income streams |
